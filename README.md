# Liver Cirrhosis Prediction

## Overview
A machine learning model to predict liver cirrhosis disease 
stages using patient medical records, integrated with an 
interactive Power BI dashboard for clinical data visualization.

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Random Forest Classifier
- Power BI

## What This Project Does
- Preprocessed and cleaned patient medical records dataset
- Built a Random Forest classification model for disease 
  stage prediction
- Evaluated model performance using accuracy metrics
- Integrated prediction results into an interactive 
  Power BI dashboard

## Project Structure
- `liver_cirrhosis_prediction.ipynb` — ML model notebook
- `README.md` — Project documentation
